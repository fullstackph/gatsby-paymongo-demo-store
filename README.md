# Gatsby + Paymongo Demo Store
* [Demo](https://fullstackph.github.io/gatsby-paymongo-demo-store/)

## Development

```bash
git clone https://github.com/fullstackph/gatsby-paymongo-demo-store.git
cd gatsby-paymongo-demo-store
yarn install
yarn start
```
